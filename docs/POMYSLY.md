# Pomysły — Protokół Urodzinowy

Luźne pomysły, które wpadają w trakcie pracy nad projektem. Bez oceniania, bez zobowiązań. Dopiero gdy pomysł dojrzeje — trafia do wymagań lub ADR.

Format wpisu:
```
### [DATA] Tytuł pomysłu
Opis. Dlaczego fajne, gdzie mogłoby pasować, co trzeba przemyśleć.
Status: surowy | do przemyślenia | odrzucony | przeniesiony do wymagań
```
